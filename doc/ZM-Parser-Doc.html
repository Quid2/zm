evalSections :: Map String (Parser a) -> String -> Either String [a] Source #

Document evaluation

Variation: Section parsers, in addition of their content are also given access to: * their context (the doc and their position in it) * the parser itself so thaty can implement nested parsing. * a read/write environment [(String,a)]

>>> data R = AChar Char | AStr String deriving (Show)
>>> parsers = M.fromList [("char", AChar <$> charLiteral), ("$", AStr <$> stringLiteral)]

>>> parsers = M.fromList [("char", Left <$> charLiteral), ("$", Right <$> stringLiteral)]
>>> evalSections parsers "char 'c'\n$ \"abc\""
